# Certificado Android App y Chacals

Aplicación Android desarrollada en Kotlin y chihuas que genera certificados personalizados.

## Características
- Certificados con escudos personalizables
- Nombre de empresa y departamento
- Firmas digitales superpuestas
- Diseño responsive que ocupa toda la pantalla

## Tecnologías
- Kotlin
- Jetpack Compose
- Android Studio
